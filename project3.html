Tech Glossary

HTML
HTML, which stands for HyperText Markup Language, is the standard markup language used for creating web pages and web applications. It is the foundational language for structuring content on the web
CSS
CSS, which stands for Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in a markup language, most commonly HTML. It is a core language of the open web platform and is standardized across web browsers according to W3C specifications.
Java
Java is a popular, multiplatform, object-oriented programming language and computing platform known for its "Write Once, Run Anywhere" principle, which allows code to run on any device that has a Java Virtual Machine (JVM). It is used to build a wide variety of applications, from enterprise software to mobile apps, and is a core technology behind many of today's digital services.
